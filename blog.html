Question-1: What are the different ways to select an element in the DOM?

  • getElementsByTagName()
  • getElementsByClassName()
  • getElementById()
  • querySelector()
  • querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML for HTML content, innerText for visible text considering styling, and textContent for raw text.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where we attach a single event listener to a parent element, which listens for events that bubble up from child elements.This allows to handle events for multiple child elements efficiently, even if they are dynamically added later.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event happens on an element and then bubbles up to its parent elements, and then to their parents, all the way to the top of the page.

Question-5: How do you create, add, and remove elements using JavaScript?

  • to Create: document.createElement()
  • to Add: appendChild()
  • to Remove: remove()
